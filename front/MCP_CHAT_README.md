# MCP 聊天界面

这是一个基于 Model Context Protocol (MCP) 的实时聊天界面，支持与 MCP 服务器进行双向通信。

## 主要特性

### 🚀 实时流式传输

- 使用 WebSocket 连接，支持真正的双向实时通信
- 基于 MCP TypeScript SDK 的 `WebSocketClientTransport`
- 自动连接管理和错误处理

### 💬 直观的聊天体验

- 类似现代聊天应用的界面设计
- 消息类型区分（用户、助手、系统）
- 实时时间戳显示
- 自动滚动到最新消息

### 🔧 丰富的功能

- **连接状态指示器**: 实时显示连接状态
- **错误处理**: 友好的错误提示和重连机制
- **Sample 运行**: 一键运行预设示例
- **清空聊天**: 快速清理聊天历史
- **键盘快捷键**: Enter 发送，Shift+Enter 换行

### 🎨 现代化 UI

- 基于 Ant Design 组件库
- 响应式设计，适配不同屏幕尺寸
- 优雅的加载状态和动画效果
- 错误消息高亮显示

## 技术架构

### 核心技术栈

- **React 18**: 现代化 React 开发
- **TypeScript**: 类型安全的开发体验
- **Ant Design**: 企业级 UI 组件库
- **MCP SDK**: Model Context Protocol 官方 SDK

### 传输层

- **WebSocketClientTransport**: 基于 WebSocket 的实时传输
- **自动重连**: 网络断开时自动尝试重连
- **会话管理**: 支持会话状态保持

### 状态管理

- React useState 进行本地状态管理
- useCallback 优化性能
- useRef 管理 DOM 引用和客户端实例

## 使用方法

### 1. 启动应用

应用启动后会自动连接到 MCP 服务器：

```
ws://localhost:{动态端口}
```

### 2. 发送消息

- 在输入框中输入消息
- 按 Enter 键发送（Shift+Enter 换行）
- 或点击发送按钮

### 3. 运行示例

点击 "Run Sample" 按钮运行预设的数据过滤示例：

```
保留空气质量AQI小于100且温度大于等于30的记录，只输出 id 和 city 字段，返回数组。
```

### 4. 管理聊天

- **Clear**: 清空所有聊天记录
- **连接状态**: 页面顶部显示连接状态

## 消息类型

### 用户消息 (User)

- 蓝色图标和背景
- 显示 "You" 标签

### 助手消息 (Assistant)

- 绿色图标和背景
- 显示 "Assistant" 标签
- 显示 MCP 工具的响应结果

### 系统消息 (System)

- 灰色图标和背景
- 显示连接状态、错误信息等

## 错误处理

### 连接错误

- 自动显示错误提示横幅
- 提供详细的错误信息
- 支持手动关闭提示

### 消息错误

- 错误消息以红色边框高亮显示
- 包含详细的错误描述
- 不会中断正常的聊天流程

## 导航

通过侧边栏菜单访问：

- **Home**: 主页介绍
- **MCP Chat**: 聊天界面

## 开发说明

### 组件结构

```
McpChat.tsx
├── 状态管理 (useState, useRef)
├── 连接管理 (connectClient)
├── 消息处理 (sendMessage, addMessage)
├── UI 渲染 (Card, Space, Button 等)
└── 事件处理 (键盘、点击等)
```

### 核心 Hooks

- `useState`: 管理聊天状态
- `useCallback`: 优化消息添加性能
- `useEffect`: 处理连接和端口监听
- `useRef`: 管理客户端实例和 DOM 元素
